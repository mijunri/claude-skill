# Database Operations Skill

处理数据库相关操作时遵循以下实践：

## 查询优化

1. **避免 N+1** - 使用 JOIN、批量查询或预加载（eager load）
2. **索引** - 为常查字段、WHERE/JOIN 条件建立合适索引
3. **分页** - 大数据集使用 limit/offset 或游标分页
4. **避免 SELECT \*** - 只选取需要的字段

## 事务与一致性

- 多步写操作放在事务中
- 注意隔离级别和死锁
- 大事务考虑拆分为小批次

## 安全

- **参数化查询** - 始终使用预处理/占位符，禁止拼接 SQL
- 敏感数据加密存储
- 连接串、凭据不入库、不硬编码

## 迁移

- 可逆的迁移脚本
- 大表变更考虑在线迁移、分批执行
- 先加新结构，再迁数据，最后删旧结构
